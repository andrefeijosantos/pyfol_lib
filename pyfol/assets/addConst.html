ProofEnvironment.addConst()

addConst(name="")

Adiciona uma nova constante ao ambiente de prova.

Argumentos:

name (opcional): Nome para a nova constante (Default: "a_i"). Para provas grandes, é encorajado o uso de nomes pequenos.

Retorno:

Objeto do tipo UserConst.

Exemplo de uso:

socrates = env.addConst(name="Socrates")

a_1 = env.addConst()

python = env.addConst(name="Python")
