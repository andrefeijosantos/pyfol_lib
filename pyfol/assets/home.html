Python First Order Logic - versão 1.0

Introdução

PyFOL é uma biblioteca implementada em Python para provas automatizadas de teoremas, baseadas em Dedução Natural. Essa biblioteca foi desenvolvida para o projeto final do curso de INF420 (Inteligência Artificial), da Universidade Federal de Viçosa (UFV). Para acessar tutoriais da biblioteca acesse a aba Documentação no topo da página.
