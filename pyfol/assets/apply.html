Pred.apply()

apply(list)

Aplica uma lista de constantes em um predicado atômico para formar uma proposição ou lista de variáveis x1, x2, x3, ..., xn em um predicado atômico para formar um predicado do tipo ∀x1∀x2∀x3, ..., ∀xn P.

Argumentos:

list: Lista de variáveis ou constantes que se deseja aplicar ao predicado.

IMPORTANTE: Nessa primeira versão, é necessário que a lista de argumentos seja inteiramente de variáveis ou de constantes.

Retorno:

Proposição ou Predicado, a depender da entrada.

Exemplo de uso (Retornando Proposições):

mortal_socrates = mortal.apply([socrates]) # Mortal(Socrates)

homem_socrates = homem.apply([socrates]) # Homem(Socrates)

Exemplo de uso (Retornando Predicados):

forall_x_homem_x = homem.apply([env.x(1)]) # ∀x Homem(x1)

env.sup(homem.apply([env.x(1)]) >> mortal.apply([env.x(1)])) # ∀x Homem(x1) ⇒ Mortal(x1)
